# Blog API

A simple RESTful API for managing blog posts, built with Node.js and Express.

## Features

- **GET** all posts or a specific post by ID
- **POST** a new post
- **PATCH** an existing post to update specific fields
- **DELETE** a post by ID

## Technologies

- Node.js
- Express
- Body-parser
